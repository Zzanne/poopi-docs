# Poopi Septic Monitoring Appliance

This repository documents the architecture, services, and data interfaces of the Poopi septic monitoring system.

## Structure

- `system_overview.md` → high-level description
- `data_flow.md` → system data paths
- `services/` → individual service documentation
- `interfaces/` → MQTT, file, and GPIO definitions
- `diagrams/` → system diagrams (Mermaid)
